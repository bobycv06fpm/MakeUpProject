# MakeUpProject
게임 리소스[아바타 및 가상 인물] 생성 AI 프로젝트

## Description
- 게임 리소스 생성을 위한 AI기반 가상 얼굴 생성 및 Face Part Layer 추출

## Env
- Based On Unity Project
- Python 3.6
- Open CV 4.3.0
- Tensorflow 2.2.0
- numpy 1.17.4
- Illustration : CartoonGAN
- dlib 19.20.0
- matplotlib 3.2.2
- Pillow 7.2.0
