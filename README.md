# MakeUpProject

## Description
- 실사 사진 바탕 화장 및 Clothes Swap 프로젝트

## Env
- Python 3.6
- Open CV 4.3.0
- Tensorflow 2.2.0
- numpy 1.17.4
- Illustration : CartoonGAN
- dlib 19.20.0
- matplotlib 3.2.2
- Pillow 7.2.0
